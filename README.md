# 🏗️ ERP SYSLAJES - Sistema para Gerenciamento de Fábrica de Lajes

Sistema ERP desenvolvido para **gerenciamento completo de uma fábrica de lajes treliçadas**, contemplando processos administrativos, financeiros, produtivos e comerciais.

---

## 📌 Sobre o Projeto

O **ERP SysLajes** é um sistema desktop construído em Delphi, com foco em controle integrado das operações de uma fábrica, permitindo maior organização, rastreabilidade e eficiência nos processos do dia a dia.

O sistema foi projetado para funcionamento **em rede**, possibilitando múltiplos usuários simultâneos.

---

## 🚀 Funcionalidades

### 👥 Gestão de Usuários
- Controle de login e autenticação de usuários

### 🧾 Cadastros
- Clientes  
- Vendedores  
- Fornecedores  

### 📦 Comercial e Pedidos
- Controle de pedidos **específico para lajes treliçadas**  
- Gestão completa do fluxo de vendas  

### 💰 Financeiro
- Contas a pagar  
- Contas a receber  
- Controle financeiro básico  

### 🏭 Produção
- Controle de produção de lajes  
- Acompanhamento dos processos produtivos  

### 📊 Estoque
- Controle de estoque de materiais e produtos  

### 🛒 Compras
- Controle de compras e fornecedores  

### 📑 Relatórios
- Diversos relatórios gerenciais  
- Geração de relatórios personalizados  

---

## 🧱 Arquitetura e Padrões

- Sistema com **telas padronizadas**  
- Organização modular das funcionalidades  
- Estrutura voltada para manutenção e expansão  

---

## 🛠️ Tecnologias Utilizadas

- **Delphi 13** – Desenvolvimento da aplicação  
- **Firebird 2.5** – Banco de dados  
- **FireDAC** – Conexão com banco de dados  
- **IBExpert** – Gerenciamento do banco  
- **FastReport** – Geração de relatórios

## 🌐 Ambiente

- Aplicação **desktop**  
- Suporte a execução em rede (**multiusuário**)
